Pour repartir de zéro avec Free Claude Code :

1. Exécute le script bash clean_free_claude_code.sh :
   bash ~/Bureau/aurora-mind/clean_free_claude_code.sh

2. Toutes les traces locales (config, .env, extensions, node_modules, .fcc, .claude, test_openrouter.py) seront supprimées.

3. Tu pourras ensuite refaire une installation propre.

⚠️ Attention : cette opération est destructive et irréversible sur les fichiers/dossiers listés.
